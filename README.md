## üìò How to Use media-query

1. **Launch the Application**  
   Once you have successfully installed media-query, open it from your Applications folder or Start Menu.

   To start syncing, connect your Nano Store by following the on-screen instructions. You will enter your store details directly into the application.

   Use the easy-to-navigate interface to set media queries for your application. The media-query tool allows you to specify how your app's state changes based on device size.
